# Taxi-Boss-Script
This script is written in lua. Its made for a roblox game Taxi boss, this script can be executed in any executor that supports lua, for example: delta, jjsploit, xeno, fluxus. etc. This code is available for anyone to use.  
To use this script you have to copy the Main Script file and paste it into your executor. If you've found any bugs please tell me.
